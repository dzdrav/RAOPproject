Predicting altruism through free pizza

Release: 1.0 (22/01/2017)
=======================================

	Predicting altruism through free pizza is project made as part of Artificial 
Intelligence college course at Faculty of Science, Department of Mathematics, 
University of Zagreb (PMF-MO Zagreb).

	Based on Kaggle's AI challenge Random Acts of Pizza, it provides programs 
which predict whether the requester will or will not receive free pizza, using 
methods of supervised machine learning.

Authors: Dinko Ždravac, Petra Martinjak
Contact: dinko.zdravac@gmail.com
	 martinjak.petra@gmail.com

Requirements
---------
- Python 3.4 or higher
- installed nltk api
- installed numpy
- installed scipy
- installed scikit-learn

File List
----------
- altruistic_requests_icwsm.pdf: Case study used as basis for the work

- kaggle_readme.txt: File from Kaggle containing description of data in dataset

- nltk_classifier.py: program based on NLTK api

- pizza_request_dataset.json: The dataset in json format

- readme.txt: This 'readme' file

- sklearn_classifier.py: program based on scikit-learn module

- enduser_documentation.pdf: user documentation 

- RAOP_presentation.pdf: presentation in PDF

- narratives (folder): A folder containing words from five narratives described 

in above paper (desire, family, job, money, student)

- files containing keywords by narrative:
	.\narratives\desire.txt
	.\narratives\family.txt
	.\narratives\job.txt
	.\narratives\money.txt
	.\narratives\student.txt
